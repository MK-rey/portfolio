# Mijung "Mia" Kim — Portfolio

A single-page portfolio for marketing, brand, and APAC partnership roles.

**Live site:** `https://<your-username>.github.io/<repo-name>/`
*(replace with your custom domain once configured)*

## Stack

Pure HTML + CSS + a tiny bit of vanilla JS. No build step, no dependencies.
Fonts loaded from Google Fonts (Fraunces, Newsreader, JetBrains Mono).

## Editing

All content is in `index.html`. Look for the section markers:

- `<!-- ══ HERO ══ -->`
- `<!-- ══ ABOUT / FEATURE ══ -->`
- `<!-- ══ WORK ══ -->`
- `<!-- ══ CAPABILITIES ══ -->`
- `<!-- ══ CONTACT ══ -->`

Find any text in the file and edit directly.

## Adding images

1. Create an `/images` folder in this repo.
2. Upload images there.
3. Reference in HTML as `<img src="images/your-file.jpg" alt="description" />`.
